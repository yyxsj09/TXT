# 誼呷尚好 EP.1 活動網站 | Yi Jia Podcast EP.1 Website

這是一個中韓雙語活動頁面，可直接上傳到 GitHub Pages。

---

## 📦 專案結構
```
/ (根目錄)
├── index.html      ← 主頁（繁中＋韓文對照）
├── .nojekyll       ← 防止 GitHub Pages 忽略靜態檔
```

---

## 🚀 上傳與啟用 GitHub Pages

1. 登入 GitHub，建立一個新的公開倉庫（名稱建議：`你的帳號.github.io`）
2. 將此 `index.html` 與 `.nojekyll` 上傳到該倉庫根目錄。
3. 前往「Settings → Pages」
4. 在 **Branch** 選擇 `main`（或 `master`），再選擇 `/root`
5. 儲存設定後，等候約 1 分鐘。
6. 你的網站將出現在：
   ```
   https://你的帳號.github.io
   ```
